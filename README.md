# DevReady

> **AI 융합 React 기반 개발자 진로·역량 통합 관리 시스템**
> *동반자가 된 AI, 취업까지 함께하다.*

2026 한국ICT인재개발원 최종 프로젝트 · 2조

---

## 프로젝트 개요

DevReady는 웹 개발 직무 취업 준비생을 위한 **AI 모의 면접·역량 성장 통합 플랫폼**이다. 기술·인성 면접을 실전처럼 연습하고, 사람 면접관 없이도 답변을 항목별로 채점해 객관적이고 즉각적인 피드백을 제공한다.

핵심은 **자체 파인튜닝한 AI 엔진**이다. 외부 API를 호출하는 방식이 아니라, 직접 튜닝·서빙한 모델에 RAG(검색 기반 지식 보강)를 결합해 IT 직무에 특화된 평가를 수행한다. 면접 답변 평가에서 끝나지 않고, 맞춤 질문과 꼬리질문, 학습 추천, 성장 대시보드, 종합 리포트, 맞춤 공고 매칭까지 하나의 흐름으로 이어진다.

- **대상 사용자** — 웹 개발 직무 취업 준비생 (기술·인성 면접 실전 연습과 객관적 피드백이 필요한 사람)
- **핵심 가치** — 객관적·즉시 피드백 (5개 항목 점수화 + 보완점 제시)
- **차별점** — 자체 파인튜닝 모델 + RAG로 직무 맞춤 평가

---

## 핵심 차별점

기존 서비스(취업포털형·코딩테스트형·영상분석형·챗봇형)는 공고·이력서 중심이거나 합/불 판정에 그쳐, **콘텐츠 + 행동 분석 + 심화 질문을 통합한 서비스가 없었다.** DevReady는 그 공백을 메운다.

- **IT 직무 특화 5축 루브릭 채점** — 자체 파인튜닝한 EXAONE 엔진으로 채점
- **대화형 꼬리질문 · STAR 분석** — 단방향 면접의 한계 해소
- **5종 멀티모달 동시 분석** — STT · 말하기 속도(WPM) · 시선 · 표정 · 침묵 구간
- **자체 서빙으로 저비용 · 한국어 특화** — 합/불이 아닌 **성장 리포트** 제공
- **올인원 흐름** — PDF 리포트 · 맞춤 공고 · 이력서 연동까지 면접 하나로 끝나지 않음

## 서비스 플로우

기본 이력서만 있으면 — **정해진 순서 없이 원하는 경로로** 시작할 수 있다. (예: 진단 후 바로 면접, 학습만 진행 등)

```
입력(기본 이력서)  →  기초 진단  →  맞춤 피드백  →  학습  →  모의 면접  →  맞춤 공고 매칭
   이력서 등록        기초지식 평가    약점 진단      이론·코딩테스트   이력서 기반 질문   수준 맞춤 회사
   직무 선택          난이도 설정      성장 대시보드   부족 지식 습득    실전 피드백        추천
```

## 시스템 아키텍처 (AI)

```
입력                FastAPI 서빙          EXAONE Deep 7.8B           구조화 출력
질문·답변      →     요청 검증·라우팅  →   4-bit + QLoRA 어댑터   →   JSON (평가·질문·리포트)
이력서·공고                                      ↕
                                          RAG · 지식 그라운딩
                                       bge-m3 임베딩 + FAISS
                                       (질문 은행 5,684개)

음성 입력(STT) → Whisper large-v3 → 텍스트 → 동일 채점 파이프라인
```

- 동시 요청은 `GEN_LOCK`으로 직렬화해 GPU 충돌 방지
- Budget Forcing(분량 상한) + 실시간 스트리밍으로 응답 지연 보완
- 모델·라이브러리 버전을 `revision` 단위로 고정해 재현성 확보
- 모델 가중치 약 5GB(4-bit 양자화)로 단일 GPU 서빙

> AI 설계·파이프라인 상세는 [AI 시스템 명세](AI시스템명세.md) 참고. *(EXAONE-Deep-7.8B는 비상업 라이선스로, 연구·포트폴리오 용도)*
